# Hospital-Integration

## Заметки
Если вдруг планируется прям запускать код, то нужные зависимости подтягиваются через composer.

Также нужен файл конфига config.php для соединения с БД.
```php
<?php
$host= 'host address';
$db = 'dbname';
$user = 'username';
$password = 'userpassword';
$debug = false;
?>
```
